# 📈 Sales Analytics Dashboard

An interactive sales analytics dashboard with real-time charts, category breakdown, and monthly revenue visualization.

## 🔗 Live Demo
👉 [https://saleanalytics.netlify.app/](https://saleanalytics.netlify.app/)

## 📸 Features

- ✅ Add and track sales with product, category, amount, and date
- ✅ Monthly revenue bar chart
- ✅ Category-wise donut chart with percentage split
- ✅ KPI cards — Total Revenue, Sales Count, Top Category, Best Product
- ✅ Filter by Today / This Week / This Month / All Time
- ✅ Data persists using localStorage

## 🛠️ Tech Stack

| Technology | Usage |
|-----------|-------|
| HTML5 | Structure & Layout |
| CSS3 | Dark theme, Animations |
| JavaScript (Vanilla) | Charts, Logic & localStorage |
